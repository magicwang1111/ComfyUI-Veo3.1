# Examples

This folder contains two small starter workflows for the four Veo nodes:

- `01_comfyui_veo31_text_workflow.json`
  Uses `ComfyUI-Veo3.1 veo-3.1-generate-preview (Text)` for a simple 4-second text-to-video request.

- `02_comfyui_veo31_image_workflow.json`
  Uses `LoadImage` plus `ComfyUI-Veo3.1 veo-3.1-fast-generate-preview (Image)` for an image-to-video request.

Prepare `config.local.json` first, then import the workflow you want to start from.
